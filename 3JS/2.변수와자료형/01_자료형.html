기본 문법

1 ) 자료형

        자바스크리브에서는 변수에 별도로 자료형을 지정해 주지 않고
        [리터럴] 에 의해 자료형이 결정된다.

        리터럴 : 변수에 대입되는 값 그 자체를 의미한다.
    
- 자료형을 확인 하는 방법

        > typeof ( 값 또는 변수명 )

- Primitive type 원시타입

하나의 값만 저장하는 자료형
  • string(문자열)

    • '',""로 묶은 데이터

  • number(숫자)

    • 정수와 실수 모두 포함

    숫자가 들어 있어도 따옴표로 묶이면 문자열로 인식

  • boolean(논리값)

    • true, false

    falsy : false로 인정 할 수 있는 값
    Boolean(0); // Boolean type 으로 형변환 형변환
    0, "" '' `` 빈문자열, NaN, underfined, null
    res = 0? 'true' : 'false'

    truthy : true 로 인정 할 수 있는 값 ( falsy를 제외한 값)

  • undefined

    • 변수를 선언만 하고 값을 할당 하지 않았을 때 변수의 초기값

  • null

    • 값이 없거나 유효하지 않은 값

- 객체

여러개의 원시 타입을 하나로 묶어 놓은 것. (배열, 함수 등)
  • 배열
    •             1. 배열 정의 하기
                let 배열명 = [값, 값, 값, .....]

                emptyArr = []; // 빈배열
                colors = ['red','blue']; // 문자열배열
                arr = [10, 'banana']; // 여러자료형을 함께 입력

            2. 배열의 값에 접근하기
                배열명[index]

                colors[0]; // red
                colors.length // 배열의 총 길이
  • function 함수
    •             
            1. 함수 정의 하기
                
                function 함수명([매개변수1, 매개변수2, ...]){
                    return res;
                }

                매개변수 : 외부로부터 입력받는 값, 생략 가능
                return : 함수의 실행결과를 반환, 생략가능


            2. 함수 호출 하기
                
                함수명()
                함수명(인수, 인수, ...);

            3. 기본매개변수(ES6 추가)

                function 함수명(num1=0, num2=0){
                    return num1 + num2;
                }
  • object 객체
    •                 1. 객체 정의 하기
                    객체명 = {
                        키 : 값,
                        키 : 값,
                        ...

                    }
                2. 객체의 값에 접근하기
                    객체명.키이름
                    객체명[키이름]


2. 데이터의 형변환

1 ) 문자열과 숫자의 '+' 연산


            문자열 + 숫자 = 문자열 

            
            숫자 + 숫자 + 문자열 = 문자열 
            (숫자가 먼저 계산되고 뒤의 문자열과 합쳐진다.)

2) 강제 형변환

내장 함수를 이용한 강제 형변환 

            Number(값), parseInt(값), parseFloat(값)
            
문자형, 논리형 => 숫자형 변환 


            
String(값), 값.toString()
            
숫자형, 논리형 -> 문자형 


            
Boolean(값)
            
문자형, 숫자형 -> 논리형 변환

3. 연산자

'==' 연산자와 '===' 연산자

'=='은 값이 일치하는지 확인하는 연산자이다.
'==='은 값과 자료형이 일치하는지를 확인하는 연산자이다.
